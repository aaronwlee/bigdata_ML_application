Project Topic:

Build, finetune and deploy a Machine Learning model application using unsupervised clustering analysis to predict geological domain units from laboratory sample geochemistry.



Dataset:

The data consists of laboratory geochemistry analysis performed on continuous rock samples obtained from drilling. These samples belong to the TUL exploration project in Manitoba. Samples and results are stored in the Assay table which will be used to create the ML model.

ASSAY TABLE
Description Laboratory results grouped by drillhole number (HOLEID) and analyzed interval (SAMPFROM - SAMPTO). It contains 96 columns with geochemical results for each sampled interval (SAMPLEID).
Size 12648 rows x 96 columns


Objective:

The main objective of the project is to build an application to support geologists with the identification of rock types (lithology) in their exploration areas. This application will (1) speed up the geological logging process and (2) provide qualitative information for the project.

To build the application, the best machine learning model will be trained and tested using a subsample of the existing dataset, using a refined pipeline and hyperparameter tuning in Spark (Databricks), to cluster future drilling samples and support the identification of areas better suited for gold exploration and mining. Once the pipeline is created, options for packaging the model and deploying it will be explored, including creating widgets on Databricks and building a front end using Flask.



Workflow:


An application will be created with the best ML model for deployment.
